# Telegram Contact Bot

Flask webhook-бот, который перехватывает контакты в Telegram-группе и спрашивает владельца, записывать ли их в Google Sheets.

## Переменные окружения

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |
| `WEBHOOK_URL` | Публичный домен (например, Railway: `https://your-app.railway.app`) |
| `OWNER_TELEGRAM_ID` | Ваш Telegram ID (число) |
| `GOOGLE_SHEETS_ID` | ID Google таблицы (из URL) |
| `GOOGLE_CREDENTIALS_JSON` | JSON строка сервисного аккаунта Google |
| `PORT` | Порт сервера (по умолчанию 8080) |

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Деплой на Railway

1. Подключите этот репозиторий
2. Добавьте все переменные окружения
3. `WEBHOOK_URL` = ваш Railway домен (например `https://my-bot.up.railway.app`)
4. Бот автоматически зарегистрирует webhook при старте

## Как работает

1. Бот добавлен в группу
2. Кто-то присылает контакт в группу
3. Бот пересылает контакт владельцу (`OWNER_TELEGRAM_ID`) с кнопками
4. Владелец нажимает **✅ Записать** — контакт вставляется в строку 4 таблицы (после строки 3), сдвигая старые записи вниз
5. Владелец нажимает **❌ Пропустить** — бот отвечает «Пропущено»

## Google Sheets

Сервисному аккаунту нужно дать доступ **Редактор** к вашей таблице.

Столбцы (строка 4 и ниже):
- A: пусто
- B: дата (дд.мм.гггг)
- C: имя контакта
- D: номер телефона
- E: пусто
